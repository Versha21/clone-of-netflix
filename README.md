# 🎬 Netflix Clone (Frontend)

A simple Netflix UI Clone built using HTML & CSS.
This project replicates the look and feel of the Netflix homepage with modern UI elements.

---

## 🚀 Features
* Netflix-style UI design
* Trending movies section
* “More reasons to join” cards
* Hover effects on movie posters
* Basic responsive layout

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📂 Project Structure

📁 netflix-clone
├── netflix.html
├── css part.css
└── assets (images)

---

## 📸 Preview

Netflix-style homepage with trending movies and feature cards.

---

## ⚙️ How to Run

1. Clone the repository
  https://github.com/Versha21/clone-of-netflix/tree/main

2. Open the folder

3. Run NETFLIX.com in your browser
